## Installation

Download and install Python and Docker on your machine.

```bash
  pip install python or pip install python3
  pip install requirements.txt
  
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/spirosbou/my_flask.git
```

Go to the project directory

```bash
  cd my-project
```
